# E-commerce Data Pipeline

This project is an end-to-end data engineering pipeline for analyzing e-commerce user events.

The goal of the project is to collect raw event data, store it in a database, transform it into analytical tables, and visualize key metrics in a dashboard.

## Tech stack
- PostgreSQL
- dbt
- Apache Airflow
- Power BI
- Docker
- GitHub

## Project scope
The pipeline processes user events such as:
- page_view
- product_view
- add_to_cart
- purchase

It allows analysis of:
- daily active users
- number of purchases
- revenue
- conversion rate

## Project status
Work in progress.

## Other
Sample data is generated for demonstration purposes and does not contain real personal information.
